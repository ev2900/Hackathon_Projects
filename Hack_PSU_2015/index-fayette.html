What We Do

Our Projects

Sexual assaults are a major problem at universities today. Unfortunately, Penn State is no exception. Sexual assault can happen at any time and to any individual. The Penn State community receives text notifications and email notifications but what does that really explain? The notifications lumps together sexual assault with burglary, simple assault, and other crimes. We at Alertpsu believe that we canâ€™t lump together these other crimes with the most serious epidemic in this Universityâ€™s history. Our solution is to help teach the Penn State Community to be proactive about keeping safe. Our project compiles data on sexual assaults to provide an accurate, unbiased and sound representation of Penn Stateâ€™s most pressing issue. We believe the present format the University uses to distribute information on sexual assaults is important, but we also believe that more needs to be done. Our website tracks data on this pressing issue and distributes it in an easy to understand format. The mobile friendly application and scalable app gives Penn State the ability to frame sexual assault on a more effective platform.

We built this to make a difference. Preventing sexual assault is the task. Awareness is a start.

How to Do It

What Technology





Data + Tech = Change



Technology can provide a solution. Our platform is a web application. It utilizes Mapbox API and Amazon Web Services to create this website. With the Mapbox API the map is easily updateable. Using Amazon Web Service makes the project scalable and redeployable.

Having an open source as well as being on a sustainable platform, any university is able to set up this same tool in only moments for no cost. Hosted on AWS , visualized with MapBox and open sourced on GitHub.
